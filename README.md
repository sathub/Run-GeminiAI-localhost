# Gemini AI Chatbot (Python + React)

This project demonstrates how to build a chatbot powered by **Google Gemini AI**, with a **Python backend** handling API calls and a **ReactJS frontend** providing a user interface. The chatbot runs locally on your machine.

---

##  Features
- Python backend using **Flask/FastAPI** to call Gemini API
- ReactJS frontend for interactive chat UI
- Secure API key management
- Localhost development setup
- Easy deployment-ready structure

---

Start the Application

python app.py

cd frontend
npm start
